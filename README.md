# Slotify css

Wrap all applicable selectors of an css sheet into ::slotted brackets.

> Please note that Slotify css is currently under development and not yet suited for production

## Installation

```shell
 $ npm i slotify-css
```

## Usage



```ts
import slotifyCss from "slotify-css"

slotifyCss()
```
